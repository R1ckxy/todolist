# TodoList - Консольное приложение

Консольное приложение для управления списком задач с возможностью планирования по дням недели и времени. Позволяет добавлять, просматривать, отмечать выполненные и удалять задачи. Данные автоматически сохраняются в JSON файл.

##  Инструкция по запуску через терминал

1. Убедитесь, что установлен Python 3.6 или выше
2. Откройте терминал в папке с проектом
3. Выполните команду:

```bash
python main.py

```

## Список реализованных функций

- **Добавление задач** — ввод заголовка, выбор дня недели и времени

- **Просмотр задач** — список с датой создания, запланированной датой и статусом

- **Отметка выполнения** — отметить задачу как выполненную по номеру

- **Удаление задач** — удаление по номеру с подтверждением

- **Задачи на сегодня** — фильтр задач на текущую дату

- **Поиск по дню недели** — фильтр задач по выбранному дню (Пн-Вс)

- **Поиск по дате** — поиск задач на конкретную дату

- **Статистика** — общее количество, выполнено, осталось, на сегодня

- **Очистка списка** — удаление всех задач с подтверждением

- **Автосохранение** — данные сохраняются в `tasks.json` при любом изменении
